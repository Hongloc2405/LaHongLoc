### Running the Application

1. **Install dependencies:**

```bash
npm install
```

2. **Start the application with Docker Compose:**

```bash
docker-compose up
```

3. **Run the application in development mode:**

```bash
npm run dev
```

4. **Run the application in production mode:**

```bash
npm run start
```
